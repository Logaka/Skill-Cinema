# Skill-Cinema

## Используемые технологии

Jetpack Compose, Kotlin, Corouines, Flow, MVI, Room, Dagger, Retrofit2, Coil, Navigation

## Результат

Мобильное приложение для поиска фильмов и сериалов, а также
управления коллекциями фильмов.
